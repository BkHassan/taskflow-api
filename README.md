# TaskFlow API

## Description
API REST sécurisée de gestion de tâches (User → Tasks)

## Tech Stack
- Java 17
- Spring Boot 4.x
- Spring Security
- JPA / Hibernate
- PostgreSQL (H2 en DEV)

## Architecture
Controller → Service → Repository

## Roadmap

- CRUD Tasks
- Roles (ADMIN / USER)
